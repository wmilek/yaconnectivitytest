Yet Another Connectivity Test

Browser version. Measures HEAD-request round-trip time to many URLs in parallel.

Browser caveats:
  • If this page is served over https://, mixed-content rules block requests to http:// URLs. Serve over http:// to exercise the full corpus.
  • Requests use mode: 'no-cors' — responses are opaque, so we time round-trips but can't read status codes.
  • Browser-level CORS/DNS failures are all reported as NetworkError; aborts as Timeout.
